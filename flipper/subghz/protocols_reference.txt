# ============================================================================
#  PROTOCOL MODULATION TYPES
# ============================================================================
#
#  OOK (On-Off Keying):
#    Simplest modulation. Carrier on = 1, carrier off = 0.
#    Most common in cheap consumer devices.
#    Flipper preset: FuriHalSubGhzPresetOok270Async or Ook650Async
#
#  ASK (Amplitude Shift Keying):
#    Similar to OOK but with amplitude levels.
#    Flipper preset: Same as OOK presets
#
#  FSK (Frequency Shift Keying):
#    Two frequencies represent 0 and 1.
#    More resistant to noise. Used in higher-end systems.
#    Flipper preset: FuriHalSubGhzPreset2FSKDev238Async
#
#  GFSK (Gaussian FSK):
#    Smoothed FSK transitions. Common in digital protocols.
#    Flipper preset: FuriHalSubGhzPreset2FSKDev476Async
#
#  Manchester:
#    Self-clocking encoding. Transition in middle of each bit period.
#    Used by Somfy, some security systems.
#
