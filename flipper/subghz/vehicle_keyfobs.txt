# ============================================================================
#  KNOWN VULNERABILITIES
# ============================================================================
#
#  1. ROLLJAM (Universal against rolling code)
#     Affects: ALL rolling code systems without counter validation
#     Method: Jam + capture two consecutive codes
#     Flipper: Read RAW on target frequency while jamming
#
#  2. ROLLBACK / ROLLING-PWN (Hyundai/Kia)
#     CVE: CVE-2022-27254
#     Affects: Hyundai/Kia 2012-2022
#     Method: Resynchronize rolling code counter by replaying old codes
#     Severity: CRITICAL â€” unlock & start without key
#
#  3. RELAY ATTACK (Keyless Go / PKES)
#     Affects: Any vehicle with Passive Keyless Entry (PKE)
#     Method: Extend key fob signal range using two relay devices
#     Range extension: 10m -> 100m+
#     Equipment needed: Two HackRF/Proxmark devices
#     NOT a Flipper-only attack (needs signal relay hardware)
#
#  4. SUBARU FIXED CODE
#     Affects: Some Subaru models (pre-2020 Forester, Outback)
#     Method: Single capture -> unlimited replay
#     Flipper: Sub-GHz Read at 315 MHz, replay
#
#  5. VW GROUP VULNERABILITY (Garcia et al. 2016)
#     Affects: VW, Audi, Seat, Skoda (2008-2016)
#     Method: Extract master key from one ECU -> compute any key
#     Secret: Only 4 master keys used globally
#
#  6. TESLA BLE RELAY (2022)
#     Affects: Tesla Model S, 3, X, Y with Phone Key
#     Method: BLE relay extends phone key range
#     NOT a Sub-GHz attack (requires Bluetooth relay)
#

# ============================================================================
#  TIRE PRESSURE MONITORING (TPMS)
# ============================================================================
#
#  TPMS sensors broadcast tire pressure data wirelessly.
#  This data includes a UNIQUE SENSOR ID = vehicle tracking.
#
#  US TPMS: 315.000 MHz
#  EU TPMS: 433.920 MHz
#
#  Flipper can READ TPMS signals:
#    - Sensor ID (unique per wheel, 32-bit)
#    - Tire pressure
#    - Temperature
#    - Battery status
#
#  USE CASES:
#    - Track specific vehicle by TPMS ID
#    - Spoof TPMS to trigger dashboard warnings
#    - Map TPMS IDs to license plates (surveillance)
