🔍 SEVERITY GUIDELINES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

CRITICAL (Immediate action required)
  • Unauthenticated RCE
  • SQL injection
  • Exposed .git with credentials
  • Admin panel without auth
  • Active credential leaks

HIGH (Fix within 7 days)
  • Weak SSL/TLS ciphers
  • Exposed admin panels
  • Outdated software (known CVEs)
  • Missing HSTS on auth pages
  • Password forms over HTTP

MEDIUM (Fix within 30 days)
  • Missing security headers
  • Directory listing enabled
  • Information disclosure
  • Verbose error messages
  • Predictable tokens

LOW (Fix within 90 days)
  • Banner disclosure
  • Unnecessary open ports
  • Missing hardening
  • Non-critical misconfigurations

INFO (Documentation/Baseline)
  • DNS records
  • Open standard ports
  • Valid SSL certificates
  • Standard configurations
